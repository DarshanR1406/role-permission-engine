# Support

Thank you for using **role-permission-engine**! We appreciate your interest and contributions.

Before opening an issue, please use the appropriate support channel below.

---

## 💬 Questions and Help

If you need help with:

- Installation
- Configuration
- Usage examples
- Integration with your project
- Best practices

please use **GitHub Discussions** instead of creating an issue.

👉 <https://github.com/darshan-raghvani/role-permission-engine/discussions>

> GitHub Issues are reserved for **bug reports** and **feature requests**.

---

## 🐞 Bug Reports

If you believe you have found a bug:

1. Search existing issues to see if it has already been reported.
2. Make sure you are using the latest version.
3. Open a new issue using the **Bug Report** template.
4. Include steps to reproduce the problem and any relevant logs or screenshots.

---

## ✨ Feature Requests

If you have an idea to improve the project:

1. Search existing feature requests.
2. Open a new issue using the **Feature Request** template.
3. Clearly explain the problem and your proposed solution.

---

## 🔒 Security Issues

Please **do not** report security vulnerabilities through public GitHub issues or discussions.

Instead, follow the instructions in our **Security Policy**:

- See: [SECURITY.md](SECURITY.md)
